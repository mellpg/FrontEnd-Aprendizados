Todos os elementos HTML podem ter atributos.

O atributo href da tag <a> especifica a URL para a qual o link direciona.

O atributo src da tag <img> especifica o caminho da imagem a ser exibida.

Os atributos width e height da tag <img> definem o tamanho da imagem.

O atributo alt da tag <img> fornece um texto alternativo para a imagem.

O atributo style é usado para adicionar estilos (cor, fonte, tamanho etc.).

O atributo lang da tag <html> declara o idioma da página.

O atributo title define informações adicionais sobre um elemento.

<h1>Os títulos devem ser usados ​​para os títulos principais, seguidos pelos <h2>títulos secundários, depois os menos importantes <h3>e assim por diante.

<h1>- Título da página
<h2>- Títulos das seções
<h3>- Subseções

Exemplo:

<h1>Travel Guide</h1>

<h2>Europe</h2>
<h3>France</h3>
<h3>Italy</h3>

<h2>Asia</h2>
<h3>India</h3>
<h3>Thailand</h3>

Links HTML - Sintaxe
A tag HTML <a>define um hiperlink. Sua sintaxe é a seguinte:

<a href="url">link text</a>
O atributo mais importante do <a> elemento é o hrefatributo `id`, que indica o destino do link.

O texto do link é a parte que ficará visível para o leitor.

Clicar no link levará o leitor ao endereço URL especificado.

Exemplo
Este exemplo mostra como criar um link para o W3Schools.com:

<a href="https://www.w3schools.com/">Visit W3Schools.com!</a>

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

Um link não visitado é sublinhado e azul.
Um link visitado é sublinhado e fica roxo.
Um link ativo é sublinhado e vermelho.
Dica: É claro que os links podem ser estilizados com CSS para obter uma aparência diferente!

O targetatributo especifica onde abrir o documento vinculado.

O targetatributo pode ter um dos seguintes valores:

_self- Padrão. Abre o documento na mesma janela/aba em que foi clicado.
_blank- Abre o documento em uma nova janela ou aba.
_parent- Abre o documento no frame principal
_top- Abre o documento em tela cheia na janela.

Links HTML - Usar uma imagem como link
Para usar uma imagem como link, basta colocar a <img> tag dentro da <a>tag:

Exemplo
<a href="default.asp">
<img src="smiley.gif" alt="HTML tutorial" style="width:42px;height:42px;">
</a>

Papel de cada parte

1. <a href="default.asp">

Cria o link.

<a> → elemento de link (âncora)

href → define para onde o link leva

default.asp → página que será aberta quando clicar

2. <img ...>

Mostra a imagem na página.

<img src="smiley.gif">


src → arquivo da imagem

alt → texto alternativo

style → tamanho da imagem

3. </a>

Fecha o link.

Tudo que estiver entre <a> e </a> se torna clicável.

Como o navegador interpreta

Estrutura lógica:

<a href="default.asp">
    imagem
</a>


Resultado:

a imagem aparece na página

clicar na imagem abre default.asp

O que é default.asp nesse exemplo

Aqui ele é apenas uma página de destino do link.
Poderia ser qualquer arquivo, por exemplo:

<a href="pagina.html">


ou

<a href="https://site.com">


O exemplo usa default.asp apenas como exemplo de página.

A <map>tag HTML define um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis. As áreas são definidas com uma ou mais <area>tags.

Tente clicar no computador, no telefone ou na xícara de café na imagem abaixo:

<img src="workplace.jpg" alt="Workplace" usemap="#workmap">

<map name="workmap">
  <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
  <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
  <area shape="circle" coords="337,300,44" alt="Coffee" href="coffee.htm">
</map>

Uma área clicável é definida usando um <area>elemento.

Forma
Você deve definir o formato da área clicável e pode escolher um destes valores:

rect- define uma região retangular
circle- define uma região circular

default- define toda a região
Você também precisa definir algumas coordenadas para poder posicionar a área clicável na imagem.

Mapa de imagem e JavaScript
Uma área clicável também pode acionar uma função JavaScript.

Adicione um clickevento ao <area>elemento para executar uma função JavaScript:

O <picture>elemento HTML oferece aos desenvolvedores web mais flexibilidade na especificação de recursos de imagem.

O <picture>elemento contém um ou mais <source>elementos, cada um fazendo referência a diferentes imagens através do srcset atributo. Dessa forma, o navegador pode escolher a imagem que melhor se adapta à visualização e/ou ao dispositivo atual.

Cada <source>elemento possui um mediaatributo que define quando a imagem é mais adequada.

Exemplo
Exibir imagens diferentes para tamanhos de tela diferentes:

<picture>
  <source media="(min-width: 650px)" srcset="img_food.jpg">
  <source media="(min-width: 465px)" srcset="img_car.jpg">
  <img src="img_girl.jpg">
</picture>

Quando usar o elemento de imagem
Existem dois objetivos principais para o <picture>elemento:

1. Largura de banda
Se você tiver uma tela ou dispositivo pequeno, não é necessário carregar um arquivo de imagem grande. O navegador usará o primeiro <source> elemento com valores de atributo correspondentes e ignorará os elementos seguintes.

2. Suporte a formatos
Alguns navegadores ou dispositivos podem não suportar todos os formatos de imagem. Ao usar o <picture>elemento, você pode adicionar imagens de todos os formatos, e o navegador usará o primeiro formato que reconhecer, ignorando os elementos seguintes.

Exemplo
O navegador usará o primeiro formato de imagem que reconhecer:

<picture>
  <source srcset="img_avatar.png">
  <source srcset="img_girl.jpg">
  <img src="img_beatles.gif" alt="Beatles" style="width:auto;">
</picture>
Nota: O navegador utilizará o primeiro <source>elemento com valores de atributo correspondentes e ignorará quaisquer <source>elementos subsequentes.

Como adicionar um favicon em HTML
Você pode usar qualquer imagem que desejar como favicon. Você também pode criar seu próprio favicon em sites como https://www.favicon.cc .

Dica: Um favicon é uma imagem pequena, portanto deve ser uma imagem simples com alto contraste.

Uma imagem de favicon é exibida à esquerda do título da página na aba do navegador, assim:

Exemplo de favicon
Para adicionar um favicon ao seu site, salve a imagem do favicon no diretório raiz do seu servidor web ou crie uma pasta chamada "images" dentro do diretório raiz e salve a imagem do favicon nessa pasta. Um nome comum para uma imagem de favicon é "favicon.ico".

Em seguida, adicione um <link>elemento ao seu arquivo "index.html", após o <title>elemento, assim:

Exemplo
<!DOCTYPE html>
<html>
<head>
  <title>My Page Title</title>
  <link rel="icon" type="image/x-icon" href="/images/favicon.ico">
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

A floatpropriedade CSS não foi originalmente concebida para alinhar <div>elementos lado a lado, mas tem sido usada para esse propósito há muitos anos.

A floatpropriedade CSS é usada para posicionar e formatar conteúdo e permite que os elementos sejam posicionados horizontalmente, em vez de verticalmente.

Exemplo
Como usar o float para alinhar elementos div lado a lado:

<style>
.mycontainer {
  width:100%;
  overflow:auto;
}
.mycontainer div {
  width:33%;
  float:left;
}
</style>
