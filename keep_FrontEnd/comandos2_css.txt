📦 1. Estrutura, Dimensões e Box Model
Estas propriedades definem o "corpo" do seu elemento (como o retângulo #gerenciador).

width: 90%: Define uma largura fluida. Ocupa 90% da largura disponível da tela, adaptando-se a celulares e tablets.

max-width: 1000px: É a trava de segurança. Impede que o elemento estique demais em monitores muito largos, mantendo a legibilidade.

min-height: 500px: Garante que o bloco tenha uma altura mínima de impacto, mesmo se estiver vazio.

width: fit-content: O elemento encolhe ou cresce para abraçar exatamente o conteúdo dentro dele (como o texto de um botão).

padding: Define o espaço interno. Afasta o conteúdo (texto) da borda, criando "respiro" dentro da caixa.

margin: Define o espaço externo. Afasta um elemento dos outros vizinhos na página.

margin: auto: O navegador calcula o espaço disponível e divide igualmente, centralizando o bloco horizontalmente.

📍 2. Posicionamento e Comportamento
Controla como o elemento se comporta em relação à página e aos outros objetos.

position: relative: O elemento segue o fluxo natural do código (fica um embaixo do outro), mas permite que você use o z-index e faça pequenos ajustes de posição sem quebrar o layout.

position: fixed: O elemento "foge" do fluxo e fica preso na tela, mesmo que você role a página (comum em Navbars).

display: flex: Ativa o modo flexível. É a ferramenta mais poderosa para alinhar itens internos com precisão.

display: inline-block: Permite que o elemento se comporte como um bloco (aceita larguras e margens), mas não força uma quebra de linha obrigatória após ele.

z-index: 1000: Define a "profundidade". Valores altos garantem que o elemento fique por cima de todos os outros.

⚙️ 3. Alinhamento com Flexbox
Propriedades usadas quando o display: flex está ativo no elemento ou no "pai" dele.

flex-direction: column: Empilha os itens internos como se fossem uma lista vertical (um embaixo do outro).

justify-content: flex-start: Alinha o conteúdo ao início do eixo (no topo, se for coluna).

align-self: center: Faz um item específico se centralizar sozinho, ignorando o alinhamento dos outros irmãos.

align-self: flex-end: Joga o item específico para o final da linha (geralmente o lado direito).

🎨 4. Estética, Visual e Identidade (Gamer/Neon)
Trabalha a beleza e o estilo visual do projeto.

background-color: #0a0a0a: Cor de fundo sólida (quase preta).

background-color: rgba(0, 0, 0, 0.7): Fundo preto com transparência (o 0.7 controla a opacidade).

border: 1px solid #00f2ff: A moldura fina em azul neon.

border-radius: Define o quão arredondados são os cantos (ex: 12px para caixas, 8px para botões).

box-shadow: Cria sombras ou brilhos (glow). No seu caso, cria a aura neon azulada.

text-decoration: none: Remove o sublinhado padrão de links, deixando o botão mais limpo.

transition: all 0.3s: Cria uma animação suave. Quando você passa o mouse, a mudança de cor ou tamanho leva 0.3 segundos para completar.

text-align: center: Garante que o texto dentro da caixa esteja perfeitamente centralizado.
