Comparativo de Estilização de Imagem

1. Apenas Object-fit

A imagem mantém a proporção, mas as quinas "vazam" do círculo.

overflow: visible;

2. Apenas Overflow

Sem object-fit: a tela do Windows fica "esmagada" lateralmente.

object-fit: fill;

3. Maneira Correta

A tela mantém a proporção original, cortando as laterais.

Combo: Hidden + Cover
